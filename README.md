# beast

BEAST is a real-time multiplayer browser game inspired by the 1984 classic.  
Each player joins the same shared world, picks a glyph/color, and tries to survive while beasts roam the map.

## How to play

- Move with **Arrow keys**, **WASD**, or **HJKL**
- On touch devices, use the on-screen directional buttons
- Avoid beasts: touching one kills you
- Push blocks to crush beasts and clear space
- The world and beast count scale as more players join

## Run locally

```bash
npm install
npm start
```

Then open `http://localhost:8080`.

## Deploy to Fly.io

```bash
./deploy.sh <fly-app-name>
```
